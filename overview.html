Implementation:

  • Once the user opens the website, login page is displayed where the user is supposed to enter the details and login if already registered.The password entered by the user is checked with the password which is stored in the database under the given user's email.


  • If the user is not registered, then one can click on signup now button and then the signup page is displayed.The details entered by the user are stored in the database and the clicking signup button takes user to the login page.


  • If login is successful, an alert box saying login successful is displayed which takes to the home page of the application and if login is not successful it is redirected back to the login page.



  • This page appears after new user sign up into catchup website.Here user will be asked to enter basic information like phone number,date of birth,address,education, professional information,hobbies.

  • Homepage: This page acts like a welcome page for the user after logging into catchup website. User can see his/her username and can go to other pages like chat page,profile edit page and logout page

  • Profile page:This page shows the basic profile information of the user.


  • Profile edit page:In this page, the user can update his/her profile information and on clicking update button it takes us to the profile page which displays the updated information.


  • Friends page:Here user can delete and add friends.All users who are in database will be shown in suggestions to make friends.


  • Chat page:Here user can chat with his/her friends listed in friends page by clicking on chat button.


  • Log out:By clicking on the logout button, one can log out of the website.
